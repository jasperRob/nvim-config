# My NeoVim Config

## Install

```
git clone --depth 1 https://github.com/wbthomason/packer.nvim\
 ~/.local/share/nvim/site/pack/packer/start/packer.nvim
```

open vim and run:
```
:PackerSync
```


to install treesitter support, run:
```
:TSInstall <languag>
```

to install language servers, run:
```
:LSPInstall <language>
```
